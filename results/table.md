## Замечания

- **Сравнение со статьёй Alchemy**: MPMP даёт MAE ~0.045 для mu. Наши модели пока хуже, но обучены на меньшем числе эпох.
- **EGNN Vector плохо учит `mu`**: mu_mae=0.71 против 0.245 у скалярного EGNN (в 2.9× хуже).
  Гипотеза: градиент через `pred.norm(dim=-1)` вырождается при `μ → 0`.
  Запланированный фикс (v32+): clamping `pred_norm = pred.norm(...).clamp(min=1e-3)`.
- **TDA в текущей реализации ухудшает метрики**: test_loss 0.285 → 0.321 для EGNN+TDA.
  Гипотеза: 52D TDA-фичи (Betti counts могут быть O(10²)) без нормализации
  доминируют над LayerNorm-нормализованным mol_emb.
  Запланированный фикс (v32+): `BatchNorm1d(tda_dim)` или FiLM-модуляция
  (модуль `src/tda/film.py` уже реализован, но не подключён).
- **Alpha лучше всех у EGNN Vector+TDA**: 0.357 vs 0.460 у EGNN.
  Возможно, векторный выход улучшает внутренние представления для alpha.

## Воспроизводимость

Все результаты получены с фиксированным seed=42:
- Stratified split по HOMO-LUMO gap
- Train/Val/Test: 162063/20257/20259 молекул (80/10/10)
- Target normalization: (y - mean) / std по train выборке
- Coordinate normalization: pos / 5.0
