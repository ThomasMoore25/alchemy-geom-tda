### Корневая причина

**54% TDA-фичей (28 из 52) — ВСЕГДА НОЛЬ** для молекул Alchemy:

| Группа фичей | Всего | Информативных | Всегда ноль |
|-------------|------:|-------------:|------------:|
| H₀ Betti curves | 16 | 16 | 0 |
| H₁ Betti curves | 16 | 4 | 12 |
| H₂ Betti curves | 16 | 0 | 16 |
| Persistence entropy | 3 | 3 | 0 |
| Mean persistence | 1 | 1 | 0 |
| **Итого** | **52** | **24** | **28 (54%)** |

**Почему H₁ и H₂ всегда ноль:**
- Alchemy молекулы маленькие (9-29 атомов) — топология простая
- H₂ (пустоты) не существует в 2D/3D молекулярных графах такого размера
- H₁ (петли) появляются только при больших радиусах фильтрации, но `max_radius=5.0` обрезает их

**Что происходит после BatchNorm1d:**
1. 28 всегда-нулевых фичей проходят через BatchNorm
2. BatchNorm нормализует к mean=0, std=1 — нули превращаются в случайный шум
3. Этот шум конкатенируется с mol_emb (128D) + global_desc (9D) = 189D
4. 28/189 = 15% входа head — чистый шум
5. Модель тратит ёмкость на игнорирование шума → хуже метрики

### Вывод по TDA

TDA **не помогает** на датасете Alchemy по объективным причинам:
1. Молекулы слишком маленькие для сложной топологии
2. 54% фичей несут ноль информации
3. BatchNorm превращает нули в шум
4. EGNN уже кодирует 3D-структуру через эквивариантные признаки — TDA избыточен

**Это валидный негативный результат.** TDA может быть полезен для:
- Больших молекул (белки, полимеры >100 атомов)
- Датасетов без 3D-координат (только графы)
- Задач, где топология важнее геометрии (классификация по форме)

## Влияние batch size

| BS | EGNN μ MAE | EGNN α MAE | EGNN gap MAE | Эпох |
|----|----------:|----------:|-------------:|-----:|
| 256 | **0.179** | **0.393** | **0.0041** | 131 |
| 512 | 0.245 | 0.460 | 0.0046 | 164 |
| 1024 | 0.286 | 0.574 | 0.0058 | 110 |

Больший batch size = меньше шагов градиента на эпоху = хуже сходимость.
Для лучших метрик используйте `--batch_size 256`.

## EGNN Vector — векторный выход

| Метрика | EGNN (скаляр) | EGNN Vector (вектор) | Разница |
|---------|-------------:|--------------------:|--------:|
| μ MAE | 0.245 | 0.710 | +190% (хуже) |
| α MAE | 0.460 | **0.390** | -15% (лучше!) |
| gap MAE | 0.0046 | 0.0041 | -11% (лучше) |

Векторный выход μ хуже сходится (zero-init зарядов), но улучшает α и gap —
векторное представление даёт более богатые признаки для других таргетов.
